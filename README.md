# ORBWin ESTABLE Cinnamon Theme
---
Windows Vista and Windows 10 FUSION
ORWin is a Cinnamon Theme with original recurses from Windows Vista, it's work with Cinnamon 3.2
Test OK -> Linux Mint 18.1 Sarah
Test OK -> Debian 9 Stretch

<img src="https://github.com/sdanielch/ORBWin-Cinnamon-Theme/raw/master/theme.png" width="100%" />

Para completar el tema es necesario descargar la última versión del applet "Configurable Menu" (DESCARGA:<a href="https://github.com/lestcape/Configurable-Menu"> https://github.com/lestcape/Configurable-Menu </a> ) y la configuración diseñada por mi para este tema, descarga "CONF-CONFIGURABLEMENU.json" de este repositorio y una vez añadido el applet reestablecer con ese fichero.
